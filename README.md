# Idle-Developers
 # Idle Developer  This project is a sample game developed using the Unity game engine. The game is made with idle mechanics The project provides a good example to understand the basic features of Unity, game development processes and most importantly how to make an idle game. You can install and use the game.  ## Features  - do certain tasks - Unlock developers to help you - Money System - Mini clicker game -And self-improvement in the game  ## Licence  This project is licensed under the MIT License.
